```
Login
POST      /login
{
  "name": "jose",
  "password": "root"
}

Cajeros
GET       /api/cajeros
POST      /api/cajero
GET       /api/cajero/{id}

DELETE    /api/cajero/{id}

Maquinas Registradoras
GET       /api/maquinasregistradoras

DELETE    /api/venta/{id}
```
